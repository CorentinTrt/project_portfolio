# Corentin's portfolio

This is my portfolio

## Usages

```shell
$ yarn install
# to install dependencies
```

```shell
$ yarn dev
# to run the project on development mode
```

```shell
$ make docker_build
# to build the docker image
```

## Deployment

The project is deployed on GCP using Cloud Run and Cloud Build. A build is triggered on a push to the `main` branch.
